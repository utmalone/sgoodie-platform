## Visual Design Notes

### Typography
- Large editorial headings for page titles
- Small uppercase labels for categories/sections
- Charcoal text color (not pure black)

### Layout
- Full-bleed images for heroes
- Consistent side padding on content sections
- Alternating layouts create visual interest

### Interactions
- Subtle hover effects on links and images
- Lightbox for image viewing
- Smooth scroll and transitions
